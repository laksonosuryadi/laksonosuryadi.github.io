Phase-0

This is my very first website, which I built from scratch using only HTML and CSS. I tried to apply all knowledge of HTML and CSS that I gained from HACKTIV8 Coding Bootcamp online preparation class. They call this class as Phase-0, which divided into 6 weeks. Each week consists of fundamental knowledges, which will be useful as a foundation for the upcoming immersive on-site coding class,later on after completing Phase-0. W E E K - 1

The main purpose of Phase-0 is to equalize the skills of the students. So that when we enter Phase-1, all of us will have the same basic knowledge and capabilty to cope with all the lessons that will be taught. We will be given some modules to read, watch, do, exercise and also some assignments to be submitted each day (from Monday to Friday). In each morning, the instructors will give us the feedback of our submitted assignments, and there are also a score board to see our achievements.

Monday, 16th January 2017 - Our First Day in HACKTIV8 Phase 0

This is the first day of Phase-0. Everyone were invited to a messenger for the geek, namely Slack. We were also assigned to study the first building block of a website, HTML! To study the basic of HTML, we use Codecademy as our main source.
Below are the assignments for Monday:

  1. Introducing ourself to the member of Slack group
  2. Completing Codecademy Course - HTML CSS Syllabus 1
  3. Completing Codecademy Course - HTML CSS Syllabus 2
  4. Make a simple HTML page

Tuesday, 17th January 2017 - Mastering HTML and CSS

After making our first simple HTML page, today let's style our webpage by using Cascading Style Sheet (CSS).
Below are the assignments for Tuesday:

  1. Completing Codecademy Course - HTML CSS Syllabus 3
  2. Completing Codecademy Course - HTML CSS Syllabus 4
  3. Completing Codecademy Course - HTML CSS Syllabus 5
  4. Style an HTML page using CSS

Wednesday, 18th January 2017 - Getting deeper into CSS

It's been 2 days since we played around with HTML and CSS. Today we will study about Box Sizing and Positioning, in order to make our page layouting and structuring skill better.
Below are the assignments for Wednesday:

  1. Completing Codecademy Course - HTML CSS Syllabus 6
  2. Completing Codecademy Course - HTML CSS Syllabus 7
  3. Completing Codecademy Course - HTML CSS Syllabus 8
  4. Completing Codecademy Course - HTML CSS Syllabus 9

Thursday, 19th January 2017 - Layouting using HTML + CSS, also learning about Version Control System using Git

We've already made a quite webpage by using the knowledge for the past 4 days. Today we were introduced to Git, a powerful tool to control our revised version of codes that we made. We were also introduced to GitHub, a platform to make our work online and saved them into our online Repository.
